# Sobre o Projeto

Este projeto faz parte do **primeiro desafio da Especialização em Data Science do Programa ONE**.

O objetivo é ajudar o **Sr. João**, proprietário da rede de e-commerce **Alura Store**, a decidir **qual das quatro lojas deve ser vendida** para que ele possa investir em um novo negócio.

Como pessoa analista de dados, realizamos uma análise comparativa baseada em métricas estratégicas para identificar a loja com menor desempenho relativo.

---

## Objetivo da Análise

Avaliar o desempenho das quatro lojas e recomendar, com base em dados, qual delas deve ser vendida.

Foram analisadas as seguintes métricas:

- **Faturamento total por loja**
- **Categorias mais e menos populares**
- **Média de avaliação das lojas recebidas pelos clientes**
- **Produtos mais e menos vendidos**
- **Custo médio do frete**

---

## Base de Dados

O projeto utiliza **quatro datasets em formato CSV**, cada um representando uma loja da Alura Store.

Cada dataset contém as seguintes colunas:

- Produto  
- Categoria do Produto  
- Preço  
- Frete  
- Data da Compra  
- Vendedor  
- Local da compra  
- Avaliação da compra  
- Tipo de pagamento  
- Quantidade de parcelas  
- Latitude  
- Longitude
